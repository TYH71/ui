# 8StarLabs Consume Registry

This skill is for downstream app work, not for maintaining this repo.

## Default context

- Next.js app
- Tailwind CSS
- shadcn initialized
- `components.json` present

## Workflow

1. Pick the right `@8starlabs/*` component for the use case.
2. Prefer registry install over rewriting from scratch.
3. Explain required imports, aliases, and usage constraints.
4. Point to examples or docs that reduce guesswork.

# 8StarLabs Component Ship

This skill is for maintainers working inside `8starlabs/ui`.

## Scope

- Component source in `registry/8starlabs-ui/blocks`
- Examples in `registry/8starlabs-ui/examples`
- Docs in `lib/docs`
- Registry/index sync and validation

## Workflow

1. Inspect the existing component, docs, examples, and registry shape.
2. Make the smallest coherent change across code, examples, and docs.
3. Keep registry metadata and generated index files aligned.
4. Run the relevant validation commands before finishing.

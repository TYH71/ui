## API Reference

| Prop             | Type                                                                          | Default     | Description                                                 |
| ---------------- | ----------------------------------------------------------------------------- | ----------- | ----------------------------------------------------------- |
| `variant`        | `"default" \| "destructive" \| "outline" \| "secondary" \| "ghost" \| "link"` | `"default"` | Optional. Controls the visual style of the button.          |
| `size`           | `"default" \| "sm" \| "lg" \| "icon" \| "icon-sm" \| "icon-lg"`               | `"default"` | Optional. Controls the button size.                         |
| `withArrow`      | `boolean`                                                                     | `false`     | Optional. Adds the animated arrow.                          |
| `arrowClassName` | `string`                                                                      | -           | Optional. Adds classes to the arrow wrapper.                |
| `asChild`        | `boolean`                                                                     | `false`     | Optional. Renders the button styles onto a child component. |
